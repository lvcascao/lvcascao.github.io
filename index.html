My Contribution — Social Sentiment Signal

The idea: Markets react to narratives. I measured them directly with a sentiment index.

  1. Collected Reddit/news posts on BTC and NVDA
  2. Scored them with FinBERT
  3. Aggregated to a rolling index
  4. Used the index as a filter/weight in my QC strategy

Build Your Own (Tutorial)

  1. Create a QuantConnect account and run the EMA crossover tutorial.
  2. Add indicators and compare against a Buy & Hold control.
  3. Run a small parameter sweep (avoid overfitting).
  4. Add an external signal (like sentiment) as a filter or weight.
  5. Paper trade first, then consider live.
  6. Track Sharpe, drawdown, and slippage—not just return.
